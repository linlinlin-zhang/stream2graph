# Workspace Archive

This directory stores files that are intentionally preserved but are not part of the active project structure.

## Current contents

- `local_probe/`
  - Local Qwen3 fine-tuning probe outputs and scratch datasets.
- `research/`
  - Temporary research artifacts such as rendered PDF inspection files.

## Rule

If a file is useful to keep for traceability but no longer belongs to the main working tree, move it here instead of leaving it in the repository root.
