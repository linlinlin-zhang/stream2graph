# Dialogue Regeneration Artifacts

This directory stores LLM reverse-generation runs for `Mermaid code -> dialogue`.

- `reports/dialogue_regen/runs/`
  - raw run outputs from local or cloud execution
  - should normally stay untracked
- `reports/dialogue_regen/published/`
  - curated bundles that are ready to be committed

The current `.gitignore` keeps raw runs out of git by default.
